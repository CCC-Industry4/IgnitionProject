2.  Overview of Project Files
    - There are four projects on the server:

    ![Open/Create Project](/images/overview/OpenCreateProject.jpg)

    - I4Project
        - The most up-to-date version of the smart home project. Use this fella to do anything important.
    - I4Project_Demo
        - A simplified version of I4Project for use with smart phones.
    - I4Project_Legacy
        - An older version of the project kept in-case of needing to revert to an older version. Hopefully with the git setup with this project will become obsolete.
    - I4Project_Test
        - An older version of the project that was used to prototype experimental features like the sandbox and object detection. Used for when multiple users were adding features to the project at once. Hopefully will also become obselete with time.
